# QPLC 步序转换工具 (Step Convert To CSV) 使用手册

## 1. 软件简介
**QPLC 步序转换工具 (Step Convert To CSV)** 是一套专为读取 PLC (可程式逻辑控制器) 内部步序 (Step) 资料并将其视觉化与汇出的软件。
本工具可透过网络连线至 PLC，读取特定的寄存器范围，将原始的步序代码解析成人类可读的动作与参数，显示于图形化介面 (GUI)，并支援将完整的步序资料汇出成 CSV 档案 (相容于 Excel)。

- **支援机型格式**: 动态扫描 `Model` 资料夹自动载入 (例如: C1M, C1K 等)
- **选择语言**: 支援繁体中文、简体中文与英文
- **开发者**: 江乙加 Eric Chiang
- **版本**: v1.1.2

## 2. 操作流程教学

1. **开启程式**: 执行 `QPLC_StepConvertToCsv.exe`。
2. **选择机型**:
   - 选择对应的设备机型。
3. **设定连线参数**:
   - 确认 PLC 的 IP 地址与 Port。
   - 点击 **[连线 PLC]** 按钮。
4. **读取步序**:
   - 确认连线状态显示为成功。
   - 点击 **[读取步序]** 按钮，系统会向 PLC 发送请求读取所有步序资料。
5. **检视资料**:
   - 读取完成后，UI 中央的步骤栏位会显示动作名称。
   - 利用「上/下」箭头按钮或输入框来翻页检视各步序。
   - 画面的列表元件亦会呈现解码后的完整资讯。
6. **汇出报表**:
   - 点击 **[汇出 CSV]** 按钮。
   - 选择储存路径与档名，将转换好的步序资料保存至电脑中。
7. **断开连线与离开**:
   - 操作完成后点击 **[断开 PLC]**，最后关闭程式即可。

## 3. 新增机型支援

本软件具备高度弹性的架构，若要支援新的 PLC 机型格式，**完全不需要修改主程式码**。

**操作步骤：**
1. 准备一份符合现有格式架构的 JSON 定义档 (例如: `NewModel.json`)。
2. 将该档案放入软件目录下的 `Model/` 资料夹中。
3. 重新启动软件，程式会自动扫描 `Model/` 资料夹，并将新机型 `NewModel` 自动载入到画面上方的机型选单中。
